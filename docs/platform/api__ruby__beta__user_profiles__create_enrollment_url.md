### Returns

- `class BetaUserProfileEnrollmentURL`

  - `expires_at: Time`

    A timestamp in RFC 3339 format

  - `type: :enrollment_url`

    Object type. Always `enrollment_url`.

    - `:enrollment_url`

  - `url: String`

    Enrollment URL to send to the end user. Valid until `expires_at`.
