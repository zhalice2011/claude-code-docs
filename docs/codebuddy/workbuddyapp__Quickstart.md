# 快速开始

> 本文帮助您在 2 分钟内完成第一个任务。

## 一、下载安装与登录

### 下载 App

- **iOS**：在 App Store 搜索「**WorkBuddy**」下载
- **Android**：在应用市场搜索「**WorkBuddy**」下载

### 登录

1. 打开 App，可选择「**手机号**」「**QQ**」「**微信**」及**其他**方式登录

![](/docs/static/quickstart-1.BGdQxvRS.png)
2. 浏览并勾选 **《用户协议》** 和 **《隐私政策》**
3. 确认授权后自动进入主页面

## 二、授权必要权限

App 会请求以下权限：

1. **相机权限** — 点击「允许」，用于拍照上传
2. **麦克风权限** — 点击「允许」，用于语音输入
3. **相册/存储权限** — 点击「允许」，用于上传图片和保存产物
4. **通知权限** — 点击「允许」，用于接收任务完成通知

> 如果误点了拒绝，可在手机系统设置中找到 WorkBuddy 重新开启。

## 三、发起您的第一个任务

1. 进入 App 后，您将看到主页面，底部为**输入栏**
2. 在输入栏中用自然语言描述您的需求，例如：

	- "帮我写一份周报"
	- "分析这份数据并生成图表"
	- "帮我生成一份 PPT"
3. 点击**发送按钮**，WorkBuddy 即开始自主执行任务

![](/docs/static/quickstart-2.Clgm0aUX.jpg)

## 四、查看执行过程

任务发送后，WorkBuddy 会自动拆解为多个步骤并逐步执行：

- 每个步骤以卡片形式展示，点击 **\>** 可展开查看详情
- 完成的步骤会标记为 **「已完成」**（绿色标签）

![](/docs/static/quickstart-3.BF-kaC1N.jpg)## 五、继续追问或下达新任务

- 在同一任务中继续输入，即可追问或调整需求，WorkBuddy 会保持上下文
- 点击顶部导航中的 **「\+」新建任务**，可发起新的独立任务

## 六、切换执行环境

输入栏上方提供环境切换入口：

| 模式 | 说明 |
| --- | --- |
| ☁️ **云端工作** | 使用云端沙箱执行任务，无需依赖个人电脑 |
| 🖥 **连接电脑** | 远程操控电脑桌面端 WorkBuddy 执行任务 |

> **注意事项：** 默认为云端工作。如需使用连接电脑，请确保电脑端 WorkBuddy 处于运行状态，且与 App 使用同一微信账号登录。
