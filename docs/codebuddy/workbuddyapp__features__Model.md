# 选择模型

## 一、入口

点击输入栏中的模型名称（如「Auto ▾」），弹出模型选择面板。

![](/docs/static/model-1.Ck4A_TNP.png)![](/docs/static/model-2.BaaTiMKw.png)## 二、可用模型一览

| 模型 | 能力标签 | 推荐场景 |
| --- | --- | --- |
| **Auto** | 推理 \+ 视觉 | 智能自动选择（默认） |
| **Hy3 preview** | 推理 \+ 视觉 | 高性能推理与视觉理解 |
| **GLM\-5\.1** | 推理 | 多步骤复杂任务，深度推理 |
| **GLM\-5\.0** | 推理 | 通用任务处理 |
| **Kimi\-K2\.5** | 推理 \+ 视觉 | 截图分析、图片转文档 |
| **DeepSeek\-V3\.2** | 推理 | 高性价比推理 |
| **DeepSeek\-V4\-Flash** | 推理 \+ 视觉 | 快速响应，兼顾视觉理解 |
| **DeepSeek\-V4\-Pro** | 推理 \+ 视觉 | 高性能推理与视觉分析 |

## 三、能力标签

| 标签 | 含义 |
| --- | --- |
| **推理** | 支持逻辑推理、任务规划与多步骤执行 |
| **视觉** | 支持图片理解（拍照、截图场景请选择此类模型） |

## 四、Credits 消耗

不同模型的 Credits 消耗有所不同。在模型选择面板中，每个模型名称下方会标注相应的 Credits 倍率（如 x0\.80、x1\.00 等），数字越低消耗越少。

> **省钱小技巧**：日常简单任务选择性价比高的推理模型（如 DeepSeek\-V3\.2、GLM\-5\.0），需要图片理解时再切换到带「视觉」标签的模型。如果不确定选哪个，使用默认的 **Auto** 即可，系统会自动匹配最优模型。
