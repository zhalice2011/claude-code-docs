# 记忆

## 什么是记忆

记忆功能让 AI Agent 根据和你的对话，自动记住你的偏好、习惯和重要信息，在不同的对话中持续提供个性化的帮助。

就像你的私人助手一样，AI 会记住：

- 你的编码风格和偏好
- 项目的技术栈和架构
- 常用的工具和命令
- 你提到的重要注意事项

## 如何使用 Memory

### 创建记忆

CodeBuddy 会自动识别并存储你的重要信息。当然，你想强调让 AI 记住某些信息时，直接告诉它：

```
请记住：我喜欢使用 TypeScript 进行开发，代码风格遵循 Airbnb 规范
```

```
记住我喜欢用中文回答问题
```
AI 会自动将这些信息保存为记忆，并在未来的对话中使用。

![alt text](/docs/static/demo.w6FDYSZS.png)

### 更新记忆

如果你的偏好或信息发生变化，只需告诉 AI 新的内容：

```
更改记忆，我喜欢用英文回答
```
AI 会自动更新相关的记忆。

![alt text](/docs/static/demo2.BvlAEVl9.png)

### 删除记忆

如果某个记忆不再需要，可以要求删除：

```
删除我喜欢英文回答的记忆
```
![alt text](/docs/static/demo3.BZtA0ygQ.png)

### 记忆的作用范围

记忆会在所有项目和对话中生效，确保 AI 始终了解你的偏好。

## 常见问题

### Q: 记忆和规则有什么区别？

**记忆** 是 AI 记住的事实和偏好（"我喜欢用 TypeScript"），通过对话创建，由 AI 自动管理。**Rule** 是你设定的行为规范（"代码必须加注释"），需要手动创建配置文件。

### Q: 记忆可以在不同项目间共享吗？

是的，记忆是全局的，会在所有项目中生效。

### Q: 记忆数量有限制吗？

没有数量限制，但建议保持在合理范围内（通常 10\-20 个），以免占用过多 Token。

### Q: AI 什么时候会使用记忆？

AI 在每次对话开始时都会加载你的记忆，并在生成回答时自动考虑这些信息。你无需手动提醒。
