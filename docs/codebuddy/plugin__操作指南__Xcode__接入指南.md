目前腾讯云代码助手已经支持在 Xcode 上安装并使用，支持在 Xcode 上开启智能辅助编码的代码补全推荐能力，助力 iOS 原生应用的开发者提高编码效率。本文将介绍如何在 Xcode 上安装腾讯云代码助手。

## 版本要求

| 系统 | 版本 |
| --- | --- |
| macOS | 12\.0以上版本 |
| Xcode | 8\.0以上版本 |

## 插件下载和安装

### .dmg 文件下载

单击 [下载](https://acc-1258344699.cos.accelerate.myqcloud.com/plugins/saas/xcode/coding-copilot-genie.dmg)，即可下载 .dmg 文件。

> 更新版本建议卸载后下载新版本

### 安装

下载 .dmg 文件完成后，直接双击进行安装，并把腾讯云代码助手的图标，拖动到右边的目录，即可完成安装，如下图：

![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/7f7ad0c6139711f0a63e5254005ef0f7.png)

### 检查更新

1. 安装好腾讯云代码助手后，可以进行**检查更新**，如下图：![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/00f45417139811f09b3252540044a08e.png)
2. 检测到新版本后，可以选择**安装新版本**或**以后再说**；如果选择安装新版本，则会自动下载插件安装包，随后跟随指引进行安装即可。

### 安装后的必要设置

安装好腾讯云代码助手后，请按以下步骤操作：

1. 登录账号。单击腾讯云代码助手右上角的**登录**，会跳转到扫码登录页，登录成功后便会显示用户名。![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/49d11bc3139811f08c275254001c06ec.png)
2. 辅助功能授权和启动服务。如下图：![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/7b5fae3f139811f0a9cd5254007c27c5.png)单击**打开辅助功能设置**，开启腾讯云代码助手按钮，如下图：![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/bb7947d1139811f0a9cd5254007c27c5.png)
3. 添加为 Xcode 扩展，如下图：![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/cde8eabb139811f0854e525400454e06.png)单击**打开扩展设置**，启用 Xcode 的扩展能力，才能接入代码补全等能力。![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/0aa767dc139911f08c275254001c06ec.png)至此，腾讯云代码助手已经可以正常使用。接下来，您就可以开启您的编码之旅。

## 插件注销

1. 在左侧侧边栏中，选择**启用插件**，然后右上角单击**注销**。![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/f69bc843323e11f0847a525400454e06.png)
2. 在弹窗中单击**确定**即可退出登录。![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/18dc95b8323f11f085ef52540099c741.png)
