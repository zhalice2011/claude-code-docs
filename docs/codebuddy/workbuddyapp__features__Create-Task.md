# 发起任务

在 WorkBuddy 移动端 App 中，您可以通过多种方式发起任务。

## 一、直接输入

在**底部输入框**中用**自然语言描述需求**，**点击发送按钮**即可。

![](/docs/static/create-1.OowvJft0.png)## 二、使用快捷入口

点击**输入框上方**的**场景标签**（如「深度研究」「文档处理」），输入框会自动填入引导提示，直接发送即可进入对应场景。

![](/docs/static/create-2.Dn0zxBc7.png)![](/docs/static/create-3.CbdjY4T9.png)## 三、从侧边栏新建

打开侧边栏 → 点击 **\+ 新建任务**，进入新的空白对话

![](/docs/static/create-4.C6Ilu2l2.png)![](/docs/static/create-5.fgPu03TU.png)## 四、通过系统分享入口发起（App 独有）

WorkBuddy 移动端 App 支持系统级分享入口：

1. 在其他 App（如浏览器、相册、文件管理器）中选中内容
2. 点击 **分享** 按钮
3. 在分享菜单中选择 **WorkBuddy**
4. 内容将自动带入输入框，直接发送即可

> 支持的分享内容类型：文字、链接、图片、文件

## 五、语音输入

点击输入栏右侧的**麦克风按钮**，开始语音输入：

1. 按住麦克风按钮开始录音
2. 松开按钮自动发送语音内容
3. WorkBuddy 会自动将语音转为文字，发起任务

> 首次使用需在系统设置中授权麦克风权限。
