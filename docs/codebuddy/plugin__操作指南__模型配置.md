# 模型配置

目前，腾讯云代码助手已经支持对话大模型的接入和切换，对话大模型包含腾讯自研的混元大模型（Hunyuan），以及 DeepSeek大模型、智谱大模型、Kimi\-K2\-Thinking 大模型和Kimi\-K2\.5 大模型。

## 腾讯支持的大模型

目前，腾讯支持的模型已经内置到腾讯云代码助手中，包含腾讯自研的混元大模型（Hunyuan），以及 DeepSeek大模型、智谱大模型、Kimi\-K2\-Thinking 大模型和Kimi\-K2\.5 大模型。
