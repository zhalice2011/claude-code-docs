# 安装和登录

本文将向您介绍 CodeBuddy IDE 如何快速下载、安装、登录、更新以及设置。

## 环境要求

| 操作系统 | 支持版本 |
| --- | --- |
| macOS | macOS 11 (Big Sur) 及以上 |
| Windows | Windows 10 及以上（不支持 Windows 7/8/8\.1） |

TIP

不满足上述要求的系统将无法启动 CodeBuddy IDE。

## 下载

1. 访问 [CodeBuddy CN](https://copilot.tencent.com/ide/) 官网，根据您电脑的处理器选择对应的版本进行下载。

![alt text](/docs/static/1%E4%B8%8B%E8%BD%BD.C46BmLke.png)

## 安装

**MAC**

将左侧安装包拖拽到右侧 Applications 中进行安装。

![2安装](/docs/static/2%E5%AE%89%E8%A3%85.BU814Edu.png)

**Windows**

1. 双击下载好的安装包进行安装。如果出现下方安装提示，选择**确定**即可，表示只为当前用户安装 CodeBuddy IDE。

![安装1-win](/docs/static/%E5%AE%89%E8%A3%851-win.BriTt2Rh.png)
2. 接着选择**我同意此协议**。

![安装2-win](/docs/static/%E5%AE%89%E8%A3%852-win.Cf1s4hmV.png)
3. 选择需要安装的位置，然后一直选择**下一步**即可。

![安装3-win](/docs/static/%E5%AE%89%E8%A3%853-win.FSnQC-fi.png)
4. 安装完成如下图所示。

![安装4-win](/docs/static/%E5%AE%89%E8%A3%854-win.CF1OxFXs.png)

## 登录

### 个人版登录

**MAC**

1. 打开 CodeBuddy IDE，点击 **登录** 使用已注册的账号进行登录。

![3登录](/docs/static/3%E7%99%BB%E5%BD%95.DceArChJ.png)

2. 跳转至登录页面，选择个人微信或手机号登录。

![4登录账号](/docs/static/4%E7%99%BB%E5%BD%95%E8%B4%A6%E5%8F%B7.BVFyQZ8R.png)

3. 登录成功并返回 CodeBuddy IDE 中开始使用。

![5登录成功](/docs/static/5%E7%99%BB%E5%BD%95%E6%88%90%E5%8A%9F.iIm7TVBG.png)  
![6登录成功IDE](/docs/static/6%E7%99%BB%E5%BD%95%E6%88%90%E5%8A%9FIDE.CkUBOXxy.png)

**Windows**

1. 打开 CodeBuddy IDE，点击 **登录** 使用已注册的账号进行登录。

![登录](/docs/static/%E7%99%BB%E5%BD%95-win.ipXFoPh0.png)

2. 跳转至登录页面，选择个人微信或手机号登录。

![4登录账号](/docs/static/4%E7%99%BB%E5%BD%95%E8%B4%A6%E5%8F%B7.BVFyQZ8R.png)

3. 登录成功并返回 CodeBuddy IDE 中开始使用。

![5登录成功](/docs/static/5%E7%99%BB%E5%BD%95%E6%88%90%E5%8A%9F.iIm7TVBG.png)  
![6登录成功IDE](/docs/static/6%E7%99%BB%E5%BD%95%E6%88%90%E5%8A%9FIDE.CkUBOXxy.png)

### SaaS企业版（旗舰版）登录

1. 打开 CodeBuddy IDE，点击 **登录** 拉起浏览器验证窗口。

![登录](/docs/static/%E7%99%BB%E5%BD%95-win.ipXFoPh0.png)

2. 选择**企业**「腾讯统一身份」登录。

![SaaS企业版登录](/docs/static/SaaS%E7%99%BB%E5%BD%95-1.q6QPbndx.png)3. 选择企业在「腾讯统一身份」配置支持的登录方式。
- **手机验证码登录：** 输入在「腾讯统一身份」添加成员时填入的手机号，获取验证码登录。

![SaaS企业版登录](/docs/static/SaaS%E7%99%BB%E5%BD%95-2.-s1QY7_s.png)- **邮箱登录：** 输入在「腾讯统一身份」添加成员时填入的邮箱地址，获取验证码登录。

![SaaS企业版登录](/docs/static/SaaS%E7%99%BB%E5%BD%95-3.iZ4moIv_.png)- **SSO登录：** 输入企业域名，进入SSO登录页完成登录。

![SaaS企业版登录](/docs/static/SaaS%E7%99%BB%E5%BD%95-4.m6EcDHXP.png)TIP

- 如果企业仅开通了**企微认证源**，将直接跳转企业微信登录页面，使用企业微信**扫码登录**即可。
- 如果同时开通了多个认证源，将跳转到腾讯统一身份SSO登录页面，使用用户名密码登录。
### 专有云企业版（专享版）登录

专享版插件请联系企业管理员获取，详情请参见[专享版快速入门](./../../enterprise/PrivateQuickstart)。

1. 打开 CodeBuddy IDE，点击 **登录** 拉起浏览器验证窗口。

![登录](/docs/static/%E7%99%BB%E5%BD%95-win.ipXFoPh0.png)

2. 拉起浏览器登录页，输入账号密码。

![专有云企业版登录](/docs/static/%E4%B8%93%E6%9C%89%E4%BA%91%E7%99%BB%E5%BD%95-1.B7TWq5k4.png)3. 登录成功，即可返回 IDE 使用。

![专有云企业版登录](/docs/static/%E4%B8%93%E6%9C%89%E4%BA%91%E7%99%BB%E5%BD%95-2._4m6Zj8J.png)## 更新

1. 点击右上方的 **账户**，在菜单选项中选择 **检查更新**。

![检查更新](/docs/static/%E6%A3%80%E6%9F%A5%E6%9B%B4%E6%96%B0.Ga-Rqxvj.png)
2. 如果有新版本，左下方会有新版本的推送，直接点击 **立即安装** 按钮进行更新即可。

## 设置

在 CodeBuddy 中，您可以根据下面的路径，在设置页中进行相关设置。左侧的导航菜单中分为 **通用**、**对话**、**Tab**、**文档**，您可以根据自己的需求进行设置。

![设置](/docs/static/%E8%AE%BE%E7%BD%AE.C8-WlTsW.png)
