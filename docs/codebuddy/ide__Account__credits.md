# Credits

## 什么是 Credits？

为合理调配高级大语言模型计算资源，CodeBuddy 采用基于 Credits 的资源配额管理机制。Credits 系用于衡量 AI 任务执行过程中的资源消耗量。

在 CodeBuddy 中，所有对话相关请求均需消耗 Credits。具体消耗数量因任务复杂度及所用高级模型而异。

## Credits 如何消耗？

- **模型 Token 定价**：不同模型消耗的 Credits 有所不同，高级模型具备更强的推理能力和更丰富的知识储备，但其输入和输出 Token 的处理单价更高，因此每处理相同数量 Token 消耗的 Credits 更多。
- **任务复杂度**：简单任务（如简短问答）消耗较少，而复杂任务（如长代码分析、多轮对话、知识库检索）需更多推理计算和 Token 处理，消耗更多资源。

## 如何扣减 Credits？

系统会优先使用个人体验版的 Credits 额度，当额度不足时，会优先使用最先到期的 Credits，包括个人专业版、个人加量包等各类资源包。
