# 快速开始

说明

本文帮助您在 2 分钟内完成第一个任务。

## 一、进入并登录

1. 在微信中搜索 **腾讯 WorkBuddy** 或扫描小程序码进入
2. 首次使用时，按提示完成微信授权登录

详细登录步骤请参考 [简介 \- 登录](./Overview#三、登录)。

## 二、发起您的第一个任务

1. 进入小程序后，您将看到主页面，底部为**输入栏**
2. 在输入栏中用自然语言描述您的需求，例如：
	- "帮我写一份周报"
	- "分析这份数据并生成图表"
	- "帮我生成一份 PPT"
3. 点击**发送按钮**，WorkBuddy 即开始自主执行任务

![直接输入](/docs/static/image18.7GFrVnrO.png)## 三、查看执行过程

任务发送后，WorkBuddy 会自动拆解为多个步骤并逐步执行：

- 每个步骤以卡片形式展示，点击 **\>** 可展开查看详情
- 完成的步骤会标记为 **「已完成」** （绿色标签）

![任务执行](/docs/static/image28.C6FRYO4b.png)## 四、继续追问或下达新任务

- 在同一任务中继续输入，即可追问或调整需求，WorkBuddy 会保持上下文
- 打开侧边栏点击 **\+ 新建任务**，可发起新的独立任务

## 五、切换执行环境

输入栏上方提供环境切换入口：

| 模式 | 说明 |
| --- | --- |
| ☁️ **云上模式** | 使用云端沙箱执行任务，无需依赖个人电脑 |
| 🖥 **本机模式** | 远程操控电脑桌面端 WorkBuddy 执行任务 |

默认为云上模式。如需使用本机模式，请确保电脑端 WorkBuddy 处于运行状态，且与小程序使用同一微信账号登录。
