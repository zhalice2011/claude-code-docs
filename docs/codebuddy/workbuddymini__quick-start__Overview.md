# WorkBuddy 小程序简介

WorkBuddy 微信小程序是 WorkBuddy 的移动端入口，您可以在手机上通过文字、语音、图片及文件下达任务。小程序支持两种执行环境：

- **云上模式**：使用云端沙箱执行任务，无需依赖个人电脑
- **本机模式**：远程操控电脑桌面端的 WorkBuddy，实现跨设备协作

## 一、核心能力

- **多模态输入**：支持文字、语音、图片、文件等多种方式下达任务
- **双模式执行**：云上模式随时可用，本机模式远程操控桌面端
- **任务管理**：创建、查看、追问、中断，全流程移动端操控
- **技能扩展**：通过 SkillHub 安装技能包，扩展 WorkBuddy 的能力

## 二、进入小程序

### 1）扫码进入

使用微信扫描下方小程序码，即可直接打开 WorkBuddy 小程序：

![小程序码](/docs/static/image1.BvkfUo9Q.png)### 2）搜索进入

1. 打开微信，下拉进入 **「小程序」** 搜索栏；
2. 搜索 **腾讯 WorkBuddy**；
3. 点击进入小程序。

**提示：**

**首次使用后，WorkBuddy 小程序会自动出现在微信「最近使用的小程序」中，后续可从下拉列表快速访问。**

## 三、登录

首次打开小程序时，系统会请求微信授权登录。

![登录页面](/docs/static/image2.DQSvp7ii.png)1. 浏览并勾选 **《用户协议》** 和 **《隐私政策》** ；
2. 点击 **「授权手机号登录」** 按钮；

![授权手机号登录](/docs/static/image3.Bnv0JK3K.png)3. 选择所需授权的手机号；

![选择手机号](/docs/static/image4.BccP2dz7.png)4. 登录成功后自动进入主页面。

![登录成功](/docs/static/image5.ijuzMrQ8.png)**注意：**

**小程序登录的微信账号需与电脑端登录的微信账号保持一致，否则无法连接到本机。**
