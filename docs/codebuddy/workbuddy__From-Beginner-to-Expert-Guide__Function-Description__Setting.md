# 系统设置

个性化设置**WorkBuddy**的语言显示、字体大小和显示模式，入口为**头像\-设置\-系统设置**。

## 语言

支持切换**中文（简体）** 和 **English**。

![](/docs/static/setting-1.nHE0f1b9.png)## 字体大小

![](/docs/static/setting-2.kPmc-9U7.png)## 显示模式

### 简洁模式

开启后将简化对话界面显示，隐藏部分装饰性元素，折叠对话中工具调用等过程信息。

TIP

默认开启简洁模式

### 非高风险自动安装

上传 Skill 后仍会显示安全检测过程；

- 非高风险**自动继续安装**
- 高风险始终需要**手动确认**

### 防休眠

开启后你的电脑系统将不会进入休眠，方便手机远程操控、自动化任务持续进行。
