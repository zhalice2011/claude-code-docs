# 专有云企业版快速入门

## 开通专有云企业版（专享版）企业

通过官网或企业后台，[前往购买页](https://buy.cloud.tencent.com/aipay?operation_type=new_buy&edition_type=exclusive)，发起专享版企业新购，购买时输入企业名称、企业 ID、超级管理员手机号等信息。

![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/9cf3aa97108c11f0854e525400454e06.png)

## 访问企业管理后台

1\.购买成功后，前往[官网](https://www.codebuddy.cn/home/)使用超管手机号登录。

![访问企业管理后台](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-1.CryI-iJ8.png)2. 专享版购买成功后需要约二十分钟的部署时间，部署完成后即可从[官网](https://www.codebuddy.cn/home/)进入企业控制台。

![官网](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-2.Cqjlos9f.png)3. 首次登录需设置超管账号初始密码，设置完成后前往企业管理后台。

![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/0a852d50151711f08c275254001c06ec.png)
4. 使用超管账号，登录企业域名管理后台。

![登录](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-3.CMucIdZi.png)选择对应企业进入后台:

![选择企业](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-4.5v4HR-Yl.png)## 打包客户端

在**企业设置\-基础设置\-产品信息**中填写信息后单击**打包插件**：

![选择企业](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-12.DsXl9oU4.png)## 录入企业成员

进入成员列表，通过手动添加或批量导入的方式，创建成员账号。

![录入企业成员](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-5.D325yNCP.png)## 开启客户端授权

导入成员完成后，可以逐个或批量开启插件授权。

![客户端授权](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-6.BstHlIcB.png)## 获取企业专用客户端

在企业后台左侧边栏选择**客户端获取**，**立即下载**所需的客户端：

![获取企业专用客户端](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-7.L-adPJZ5.png)或者在**组织概览**最下方快速下载所需的客户端到本地。点击**获取全部客户端**可跳转至完整客户端列表：

![获取企业专用客户端](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-8.rtQWnG32.png)### 分享企业专用客户端

1. 在企业后台进入**客户端获取**，点击**分享客户端**：

![分享企业专用客户端](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-9.DdUa5CmQ.png)2. **复制链接**，将弹窗中的信息分享给企业成员：

![分享企业专用客户端](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-10.CgNztJPI.png)3. 成员可通过链接打开下图页面，点击所需的客户端**立即下载**：

![分享企业专用客户端](/docs/static/%E4%B8%93%E7%94%A8%E4%BA%91%E4%BC%81%E4%B8%9A%E7%89%88%E5%BF%AB%E9%80%9F%E5%85%A5%E9%97%A8-11.CMk_TXkn.png)## 成员登录客户端
