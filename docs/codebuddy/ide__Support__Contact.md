# 联系我们

使用CodeBuddy的过程中遇问题可以向 **[codebuddy@tencent.com](mailto:codebuddy@tencent.com)** 发送邮件咨询。

如果您希望CodeBuddy提供更好的功能来满足您的需求，请在**CodeBuddy IDE**内反馈给团队。

## 反馈入口

在IDE内点击右上角的头像图标，选择**帮助与反馈**：

![](/docs/static/contact-1.kmLLvOJY.png)在**意见反馈**处描述您的需求，建议上传图片并勾选上传日志，以便团队更准确地理解您的需求和问题:

![](/docs/static/contact-2.DeC4EwWt.png)
