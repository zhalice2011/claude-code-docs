在微信小程序开发者工具 IDE 中，腾讯云代码助手已经支持多种对话大模型，对话大模型包含腾讯自研的混元大模型（Hunyuan），以及 DeepSeek大模型、智谱大模型、Kimi\-K2\-Thinking 大模型和Kimi\-K2\.5 大模型。

## 腾讯部署的对话模型

腾讯部署的模型已经内置到腾讯云代码助手中，包含腾讯自研的混元大模型（Hunyuan），以及 DeepSeek大模型、智谱大模型、Kimi\-K2\-Thinking 大模型和Kimi\-K2\.5 大模型。

## 对话模型切换

腾讯云代码助手侧栏打开技术对话功能，在对话框右边的模型选项中，包含了腾讯内置的官方大模型，默认是混元大模型（hunyuan）。您可以自由切换对话模型，帮助提升您的编码体验。
