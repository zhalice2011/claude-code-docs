# 智能体模式

智能体模式是 CodeBuddy IDE 中面向复杂任务协作的一套工作方式。您可以直接在 IDE 内创建任务、持续对话、查看执行结果，并在同一个界面中完成文件检查、代码审查和结果预览。

## 适合的场景

- 需要分步骤推进的开发任务
- 需要边聊边改、持续追问的任务
- 需要查看产物、文件、变更和预览结果的任务
- 需要同时处理多个任务的场景

1. [快速开始](./Quickstart)  
 先了解如何进入智能体模式、发起第一个任务，以及界面的基本结构。
2. [创建任务](./Create-Task)  
 了解如何选择工作模式、填写任务内容和补充上下文。
3. [任务管理](./Task-Management)  
 了解如何查看任务列表、按状态筛选，以及继续处理已有任务。
