公司网络可能会限制直接访问公共网络，如有需要，可以通过以下操作配置代码助手的网络代理。

## VS Code

1. 打开菜单，选择**高级设置**。

![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/ac09139e112411f09b3252540044a08e.png)
2. 在设置项中，找到网络代理方式，可选使用系统网络代理或手动配置代理。

![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/e4e5f5c7112411f09b3252540044a08e.png)
3. 若手动配置代理，需填写代理地址。

![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/097e6b88112511f0aaa3525400e889b2.png)

## JetBrains IDEs

1. 打开菜单，选择**打开设置**。

![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/3d316aae112511f09b3252540044a08e.png)
2. 选择网络代理模式，若为手动设置，则需填写网络代理地址。

![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/54351c4d112511f09fca52540099c741.png)
