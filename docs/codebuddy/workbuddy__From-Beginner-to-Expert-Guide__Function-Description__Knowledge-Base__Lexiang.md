# 乐享知识库

腾讯乐享是一站式企业社区与智能知识管理平台，承载团队的培训、文档、问答等共享知识资产。WorkBuddy 接入后可按团队空间检索并引用乐享中的内容。

## 连接

1. 在左侧边栏点击**资料库**，选择**乐享知识库**。

![](/docs/static/knowledge-base-18.Cy1mJrGd.png)2. 点击**立即前往授权**后使用 微信扫码登录 / 手机号登录 乐享知识库。首次登录需要绑定手机号。

![](/docs/static/knowledge-base-20.1fpaDHHe.png)3. 点击**继续**完成授权，WorkBuddy 将获得以下乐享知识库权限：

![](/docs/static/knowledge-base-19.Cw4C9NPk.png)- 读取知识库权限
- 搜索和查询知识内容
- 创建和编辑知识内容

TIP

WorkBuddy 连接资料库的乐享知识库后将获得**登录用户**的乐享知识库的权限，详细授权信息见上方内容。

## 使用

### 任务引用指定文件

在任务中引用乐享知识库内容的方式有两种：

1. **添加到任务：** 在资料库 \- 乐享知识库中将当前知识库添加到任务，或选中对应文件，点击**添加到任务**。

![](/docs/static/knowledge-base-21.BdRZdP93.png)2. **上传文件：** 在新建任务的输入框上方**上传文件**图标处打开乐享知识库，支持勾选整个团队空间、知识库或选择文件，点击**确定**添加到对话。

![](/docs/static/knowledge-base-22.BnFiIzbJ.png)![](/docs/static/knowledge-base-23._c29B8_g.png)### 产物存回乐享知识库

在右侧产物中找到需要的文件，点击上传到云端，选择**乐享知识库**储存。保存到乐享知识库支持按照**团队空间**储存。

![](/docs/static/knowledge-base-8.BjSgiNFP.png)## 解除绑定

如果需要换绑账号或关闭连接，可以点击**乐享知识库**，在右上角点击**解除绑定**。
