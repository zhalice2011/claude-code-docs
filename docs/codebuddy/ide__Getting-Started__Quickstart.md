# 快速开始

本文将为您介绍如何快速打开或创建一个项目。

## 打开或创建项目

- 打开 CodeBuddy IDE，右上角图标分别对应终端和右侧聊天窗口打开与关闭。

![打开IDE](/docs/static/open-IDE.DUDuueFq.png)
- 根据实际情况选择创建一个新的项目，或打开已存在的项目。

	- 创建新项目 点击 **新建文件夹** ,支持在系统用户目录中创建一个新项目。![打开新项目](/docs/static/open-new.Ck248PNX.png)

	- 打开已存在的项目 点击 **打开文件夹**，将会在本地打开一个已存在的项目。![打开已存在的项目](/docs/static/open-exit.tqC_cv2Z.png)

	- 克隆远程 Git 仓库项目
	
	
		1. 确保已安装Git，可使用下方命令查看版本。如未安装，请访问 [Git](https://git-scm.com/downloads) 进行安装。
		
		bash
		```
		git --version
		```
		2. 获取远程 Git 仓库的 URL，例如：
		
		bash
		```
		https://github.com/leaferjs/leafer-draw.git
		```
		3. 点击 **克隆 Git 仓库**，输入 URL，然后选择克隆项目存放的目录即可。![Git clone](/docs/static/git-clone.BON75PzO.png)

至此，您就可以开启编码之旅。
