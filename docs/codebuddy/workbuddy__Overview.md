# WorkBuddy 简介

WorkBuddy 是腾讯出品的全场景 AI 办公工作台。说出要求、开始执行任务、交付完整成果。完美连接腾讯办公生态，你的办公好搭子。

## 一、核心能力

- **理解自然语言**：用一句话下达任务，无需复杂的操作步骤
- **自主规划执行**：自动拆解任务、规划步骤、执行操作
- **多模态任务处理**：支持文档、表格、PPT、数据分析等多种任务类型
- **本地文件操作**：可读取授权的电脑文件夹，进行批量处理

## 二、适用场景

- 文档生成：生成工作报告、技术文档、会议纪要等
- 数据分析：上传数据文件，自动分析和可视化
- PPT/报告生成：从需求描述生成演示文稿和报告
- 深度研究：对复杂问题进行深度调研并输出报告
- 邮件编辑、周报生成等办公场景
- 批量文件处理：自动整理、重命名、转换文件格式

## 三、与传统 AI 对话的区别

| 传统 AI 对话 | WorkBuddy |
| --- | --- |
| 只能对话，提供建议 | 能够实际执行任务 |
| 需要手动操作文件 | 自动操作本地文件 |
| 单步骤简单任务 | 多步骤复杂任务 |
| 输出文字回复 | 交付可验收的结果 |
