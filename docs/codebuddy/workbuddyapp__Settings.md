![](/docs/static/settings-1.XA08hPjm.png)## 账号设置

### 注销账号

#### 注销规则

1. 提交注销后，账号进入注销状态并保持**15天**；此期间如需恢复，需联系客服。
2. 15天状态结束后，账号正式注销，所有数据**不可恢复**。
3. 若有**未用完的付费资源**，无法完成注销。
4. 提交注销即表示已知悉并接受注销后果。

#### 注销后果

1. **数据删除：** 任务、文件、知识库、对话记忆、定时任务、AI产出文件、记忆库等数据永久删除，不可恢复。
2. **订阅服务终止：** 免费订阅服务终止；剩余积分、额度、时长、AI助手使用次数等资源不予折现。
3. **绑定关系解除：** CodeBuddy、企业微信、API密钥、第三方授权等绑定关系全部解除，无法找回。
4. **重新注册限制：** 若需用同一手机号重新注册，需在提交注销申请30天后才能操作。

#### 操作指引

1. 进入**账号设置**，选择**注销账号**：

![](/docs/static/settings-2.CBrNqkZm.png)
2. 在跳转页中点击**申请注销**：

![](/docs/static/settings-3.DMr9q9S0.png)
3. 请阅读 WorkBuddy 账号注销协议，阅读完成后，点击**同意注销协议并确认注销**：

![](/docs/static/settings-4.Caxwc4Jh.png)
4. 完成验证绑定手机号后，点击**确认注销**，账号将在30天内完成注销：

![](/docs/static/settings-5.Br_EBX1Y.png)

## 记忆库

> **记忆**让 WorkBuddy 记住你的偏好和习惯，对话越多，它就越懂你。

**我的记忆**每晚重新生成，点击可查看、编辑记忆内容。通过对话的方式告诉 WorkBuddy 要记住或忘记什么。

![](/docs/static/settings-6.DZ0ICu5L.png)## 隐私政策、服务协议及各项清单

在这一部分中，包含**隐私**、**服务协议**、**隐私政策**、**个人信息收集清单**和**第三方信息共享清单**，点击各自卡片，可以查看对应的协议内容

![](/docs/static/settings-7.CZFXwyFI.png)## 主题和语言

| 选项 | 说明 |
| --- | --- |
| **主题** | 切换浅色/深色/跟随系统 |
| **语言** | 切换界面语言（跟随系统/简体中文/繁体中文/英文） |

## 消息通知设置

点击消息通知设置，可跳转到手机系统对于 WorkBuddy 的消息通知设置策略

## 其他设置

| 选项 | 说明 |
| --- | --- |
| **分享日志** | 可以按时间范围导出日志文件（最近1小时/24小时/3天/全部日志） |
| **清除日志** | 点击可删除所有本地日志文件，请注意，此操作不可恢复 |
| **意见反馈** | 点击可反馈意见 |
| **退出登录** | 退出当前账号 |
