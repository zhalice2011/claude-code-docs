# 简介

WorkBuddy 移动端（App）是桌面端 WorkBuddy 的移动配套产品。您可以在手机上通过**文字、语音、图片及文件**下达任务，让 AI 随时随地为您的灵感加速。

## 核心能力

| 能力 | 说明 |
| --- | --- |
| **多模态输入** | 支持文字、语音、图片、文件等多种方式下达任务 |
| **双模式执行** | 云端工作随时可用，连接电脑远程操控桌面端 |
| **任务管理** | 创建、查看、追问、中断，全流程移动端操控 |
| **技能扩展** | 通过 SkillHub 安装技能包，扩展 WorkBuddy 的能力 |

---

## 执行模式

WorkBuddy 移动端（App）支持两种执行模式：

- **☁️ 云端工作**：使用云端沙箱执行任务，无需依赖个人电脑。适合通勤、差旅、周末等没带电脑的任何时刻。
- **🖥 连接电脑**：远程操控电脑桌面端的 WorkBuddy，实现跨设备协作。适合已经离开工位，但需要电脑上的 WorkBuddy 执行任务的场景。

## 进入 App

### 应用商店下载

- **iOS 用户**：在 App Store 搜索「**WorkBuddy**」下载安装
- **Android 用户**：在主流应用市场（华为应用市场、小米应用商店、应用宝等）搜索「**WorkBuddy**」下载安装

## 登录

### 登录流程

1. 首次打开 App 时，可以选择「**QQ**」、「**微信**」或「**其他**」方式授权登录
2. 浏览并勾选 **《用户协议》** 和 **《隐私政策》**
3. 点击 **「授权登录」** 按钮
4. 授权成功后自动进入主页面

> **⚠️ 注意**：App 登录的微信账号需与电脑端登录的微信账号保持一致，否则无法使用连接电脑。

### 首次启动权限授权

首次安装后，App 会依次申请以下权限，建议全部允许以获得完整体验：

| 权限 | 用途 | 建议 |
| --- | --- | --- |
| **相机** | 拍照上传、扫码 | 允许 |
| **麦克风** | 语音输入 | 允许 |
| **存储/相册** | 上传图片和文件、保存产物 | 允许 |
| **通知** | 任务完成提醒、自动化通知 | 允许 |
