# 选择模型

## 一、入口

点击输入栏中的模型名称（如「GLM\-5\.0 ▾」），弹出模型选择面板。

![模型选择面板](/docs/static/image21.DzeIrZOL.png)**注：**

**当目前任务的第一句话发送后，模型选择才可显现出来。**

## 二、可用模型一览

| 模型 | 能力标签 | Credits 消耗 | 推荐场景 |
| --- | --- | --- | --- |
| **GLM\-5\.0** | 推理 | x0\.80 | 多步骤复杂任务，深度推理 |
| **GLM\-5\.0\-Turbo** | 推理 | x0\.95 | 高性能推理，响应更快 |
| **GLM\-4\.7** | 推理 | x0\.23 | 日常任务，性价比高 |
| **MiniMax\-M2\.7** | 推理 \+ 视觉 | x0\.26 | 图片理解、拍照识别 |
| **MiniMax\-M2\.5** | 推理 | x0\.18 | 轻量任务，成本最低 |
| **Kimi\-K2\.5** | 推理 \+ 视觉 | x0\.45 | 截图分析、图片转文档 |

## 三、能力标签

| 标签 | 含义 |
| --- | --- |
| **推理** | 支持逻辑推理、任务规划与多步骤执行 |
| **视觉** | 支持图片理解（拍照、截图场景请选择此类模型） |

## 四、Credits 消耗

数字越低消耗越少。例如 x0\.18 的模型比 x0\.95 的模型便宜约 5 倍。
