# 自动化

自动化让你可以为任务设定执行时间，WorkBuddy小程序按时自动执行。

提示

以下内容是在 **Cloud（云上）** 模式下的配置指引，**Local（电脑）** 模式下 **自动化**用于接收电脑端自动化任务执行结果。

## 配置

### 添加定时任务

点击右下角`+`图标，添加定时任务：

![添加定时任务](/docs/static/auto-1.B6C45_g9.png)### 描述定时任务

1. 填写任务标题、提示词、实行频率以及执行时间，**支持选择不同模型和技能(Skills)**。

![描述定时任务](/docs/static/auto-2.C9t7YLjf.png)2. **用对话方式创建**定时任务，点击跳转至小程序主页输入任务详情：

![对话创建定时任务](/docs/static/auto-3.Bs0wNjys.png)![添加定时任务](/docs/static/auto-4.CAe3YHWa.png)3. 创建完成后，定时任务将自动添加到列表中：

![定时任务列表](/docs/static/auto-5.2KvMm1Mk.png)### 定时任务管理

点击任务右上角，可以对任务进行**编辑**、**测试运行**、**暂停**、**删除**等操作：

![定时任务列表](/docs/static/auto-6.w2aWa1dC.png)点击任务跳转**任务详情**查看测试执行结果：

![定时任务列表](/docs/static/auto-7.DdHv9DUW.png)![定时任务列表](/docs/static/auto-8.DSTvwtd9.png)
