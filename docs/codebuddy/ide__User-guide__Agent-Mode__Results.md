# 结果查看

## 右侧结果区

任务执行过程中，右侧会显示结果区，用来查看智能体产出的文件、代码变更和预览内容。您不需要离开当前任务，就可以一边看对话，一边查看结果。

当前结果区主要提供四类内容：

- **产物**：查看任务生成的文件和交付物
- **全部文件**：查看当前工作空间中的文件内容
- **变更**：查看本次任务带来的代码修改
- **预览**：查看网页或可直接打开的结果页面

这些内容都集中在同一块区域中，方便您来回切换和对照。

## 产物

当任务生成文件、表格、文档或其他可交付内容时，您可以在 **产物** 中统一查看。

产物区通常包含两部分：

- 左侧是当前任务生成的产物列表
- 右侧是选中产物的预览区域

适合在这里查看的内容包括：

- 文档类结果
- 表格类结果
- 自动生成的交付文件

![产物区示例](/docs/static/artifacts-view.HJSC1a34.png)

### 表格预览

如果任务生成的是表格文件，您可以直接在结果区中打开查看，不需要先下载再用外部工具打开。

这样更适合快速确认：

- 表头是否正确
- 数据是否完整
- 排版和格式是否符合预期

![表格预览示例](/docs/static/spreadsheet-preview.F2IaDRmv.png)

### 文档预览

如果任务生成的是文档文件，您也可以直接在结果区中查看内容。这样在交付前，可以先快速检查版式、标题和正文内容是否符合预期。

![文档预览示例](/docs/static/document-preview.C67nWXl9.png)

## 网页预览

如果任务生成的是网页、页面原型或本地启动的 Web 应用，您可以在 **预览** 中直接查看效果。

预览区通常会提供：

- 当前页面地址
- 刷新入口
- 页面内容的实时查看

这适合用于确认页面结构、样式和交互结果是否符合预期。

![网页预览示例](/docs/static/webpage-preview.5roS4B5L.png)

## 全部文件

**全部文件** 用于查看当前工作空间中的文件内容，适合在不离开任务的情况下直接浏览代码或产出文件。

这一视图通常包含：

- 左侧文件列表或文件树
- 右侧文件内容区域
- 顶部已打开文件的切换栏

适合用来：

- 快速查看某个生成文件的完整内容
- 对照不同文件之间的关系
- 检查任务是否写入了正确的位置

![全部文件示例](/docs/static/all-files-view.DzoUe72r.png)

## 变更

**变更** 视图用于查看当前任务带来的代码修改，方便您在接受结果前先做一轮检查。

这一视图通常会显示：

- 本次修改涉及的文件列表
- 每个文件的改动情况
- 选中文件后的具体变更内容

如果任务涉及代码开发、脚本生成或配置调整，建议优先在这里确认改动是否符合预期。

![变更视图示例](/docs/static/code-changes-view.CeiLIWSI.png)
