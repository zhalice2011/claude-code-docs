# 套餐管理

## 手动续期

在[官网](https://www.workbuddy.cn/)的**个人主页**进行手动续期，详细步骤如下：

1. 登录[WorkBuddy官网](https://www.workbuddy.cn/)，从右上角头像处进入**个人主页\-套餐管理**。
2. 在当前套餐右侧点击**续费**，点击**确认并支付**后使用**微信扫码**完成支付。
3. 支付成功，您的订阅将延长一个月。

## 自动续费

自动续费是指您所购买的套餐每次即将到期时，系统将自动从您签约的账户中扣取下一续费周期的费用。

在[官网](https://www.workbuddy.cn/)的**个人主页**开启自动续费，详细步骤如下：

1. 登录[WorkBuddy官网](https://www.workbuddy.cn/)，从右上角头像处进入**个人主页\-套餐管理**。
2. 在**自动续费设置**右侧点击**开启自动续费**，在弹窗中使用**微信扫码签约**完成设置。

## 套餐说明

- **自动续费：** 连续包月/连续包年默认自动续费，可在个人主页取消。
- **到期处理：** 到期后降级为体验版，已生成内容不受影响，高级功能受限。
- **暂不支持升降配：** 暂不支持有效期内直接升/降配；如需换档，需在满足退款条件时退订后重新购买。
- **最大购买期限：** 最长支持购买 **30个月**，达上限时现有权益不受影响。

## 退款指引

WorkBuddy 订单支持通过腾讯云官网发起工单退款，具体操作步骤如下：

1. 登录腾讯云官网，进入[腾讯云\-提交工单](https://console.cloud.tencent.com/workorder/category)页面，在**产品分类**处找到需要退款的产品。
2. 填写问题详情和联系方式后**提交工单**。
3. 提交退款工单后，工单将进入审核阶段，腾讯云客服人员将会在 **3个工作日** 内处理您提交的申请。
4. 工单审核通过后，系统将执行退款操作，退款将在 **5\-10 个工作日** 内原路退回。
5. 可在[腾讯云\-订单管理](https://console.cloud.tencent.com/expense/deal)页面查看退款成功的订单，订单状态为**已退款**；退款到账的款项在[腾讯云\-主页](https://console.cloud.tencent.com/expense/overview)查看。
6. 若审核不通过，您可在[腾讯云\-我的工单](https://console.cloud.tencent.com/workorder)里查看审核结果。

### 退款规则

- 同一账号下的每个商品订单仅有 **1次** 退款权益。
- **月付会员（含自动续费）：** 自订单实际支付后**24小时内**且未使用任何订单内积分（含订单下活动赠送积分），可申请退款。
- **年付会员（含自动续费）：** 自订单实际支付后**7个自然日内**且未使用任何订单内积分（含订单下活动赠送积分），可申请退款；当前定价方案生效前已完成订阅的年付会员不支持退款。
- **加量包、兑换码（WorkBuddy 提货券）** 不支持退款。

说明

- 上述积分包括套餐赠予积分和套餐基础用量。
- 更多细则请以[订阅协议](https://rule.tencent.com/rule/202606290002)原文为准。
