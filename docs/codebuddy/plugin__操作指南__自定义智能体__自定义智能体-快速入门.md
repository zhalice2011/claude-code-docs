## 智能体介绍

智能体（Agent）是一种基于大语言模型驱动的自主系统，旨在通过智能化能力提升软件开发效率和质量。对于用户需要扩展腾讯云代码助手的原子能力，我们提供两种创建智能体的方式，满足用户不同的场景需求：

- 用提示词创建智能体：无需开发，仅配置智能体提示词，可快速扩展，适合靠提示词工程的扩展方案。
- 自定义服务接入：需开发插件和工作流业务服务，部署后接入，适合有第三方业务接入、工作流编排的扩展方案。

本文将为您介绍如何用提示词快速创建并体验一个智能体。

## 用提示词创建智能体

1. 登录[企业后台管理](https://copilot.tencent.com/admin/overview)端，选择**智能体**，然后单击**新建智能体**。 ![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/82acec30692811f08eae52540044a08e.png)
2. 配置智能体，其中类型选择**用提示词创建智能体**。 ![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/aeeac63b692811f0a1c55254005ef0f7.png)

填写智能体相关信息如下：
- **智能体名称**：智能体的名称。 注意：智能体的名称不能重复，包括不能与代码助手内自带的智能体名称重复，以及不能与企业内的智能体名称重名。
- **智能体描述**：智能体的描述，可展示在本地 IDE 中。
- **提示词描述**：填写提示词，规范智能体如何更好地为您服务，提示词的质量直接影响对话输出的质量和您的体验。
- **上下文**：

	- **添加知识库**：如果功能启用后，调用指令即可关联知识库。
	- **引用文件**：如果功能启用后，调用指令会添加当前文件为上下文。
- **触发区**：默认仅在聊天区触发。
- **可见范围**：可以设定成员的使用权限，包括企业内**所有成员**和**部分成员**。当选择**部分成员**可见时，可直接单击**添加成员**按钮去进行添加，支持全选和根据 ID、姓名、账号进行输入搜索选择，如下图所示。 ![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/4d074a04692911f094035254001c06ec.png)
3. 信息填写完成后，单击**保存**，例如： ![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/11ee77e9692a11f0a1c55254005ef0f7.png)
4. 智能体创建完成后，默认已经启用智能体。 ![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/4400393b692a11f09d77525400bf7822.png)
5. 启用智能体后，打开本地 IDE 即可在对话框中通过 **@ Add** 来查看并体验智能体。 ![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/43ab970a692c11f0a1c55254005ef0f7.png)

> **说明：**

> 建议重启一下 IDE。

## 智能体管理

1. 在**智能体**列表，可以对智能体进行**启用/关闭**、**编辑**和**删除**的操作。 ![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/64235112692a11f0bcc652540099c741.png)
2. 在**编辑智能体**中，可以对所有成员和部分成员进行可见范围的设置。 ![](https://write-document-release-1258344699.cos.ap-guangzhou.tencentcos.cn/100039847436/8623b626692a11f0a1c55254005ef0f7.png)
