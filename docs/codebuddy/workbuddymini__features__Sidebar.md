# 侧边栏

## 概览

进入小程序主页，点击左上角菜单（≡）展开侧边栏：

![侧边栏](/docs/static/sidebar-overview.DEVt6YwX.png)## 详情

### 模式选择

默认**云上**模式，可切换**电脑**模式：

![侧边栏](/docs/static/siderbar-select.O1iIjvTQ.jpg)### 新建任务

跳转主页面[发起任务](./Create-Task)。

### 自动化

配置和管理自动化任务。

### 专家

按行业分类浏览专家，快速找到合适人选，支持**分享**。

### 任务

查看历史任务。点击任意任务可切换到该任务的对话页面继续处理（当前任务呈现浅绿色底色）。

### 用户信息

显示当前登录的微信用户头像与昵称，点击右侧 **⚙ 齿轮图标** 进入[设置](./Settings)。
