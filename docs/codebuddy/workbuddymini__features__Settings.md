# 设置

打开左侧边栏，点击用户名称旁的 **⚙ 齿轮图标**，进入设置页面：

![设置页面](/docs/static/settings-1.C4uquUtQ.png)## 切换登录账号（换绑手机号）

1. 进入**账号设置**，点击**换绑**拉起绑定手机号弹窗：

![设置页面](/docs/static/settings-2.CL_2cMu4.png)选择**使用其他号码**：

![设置页面](/docs/static/settings-3.D5rsFCn5.png)在跳转页中完成添加手机号：

![设置页面](/docs/static/settings-4.BCfoovPW.png)确定后还需要完成**授权**：

![设置页面](/docs/static/settings-5.BkIZtnIw.png)3. 完成后在侧边栏可以看到新的账号信息：

![设置页面](/docs/static/settings-6.Xb5dqVoO.png)## 记忆库

> 记忆让 WorkBuddy 记住你的偏好和习惯，对话越多，它就越懂你。

**我的记忆**每晚重新生成，点击可查看、编辑记忆内容，通过对话的方式告诉WorkBuddy要记住或忘记什么。

![](/docs/static/settings-7.jvOaGylZ.png)## 我的分享

在**我的分享**中，可以查看已分享的文件详情：

![](/docs/static/settings-8.dmko5Jv7.png)点击卡片进入**分享详情**查看产物分享的创建日期、有效期：

![](/docs/static/share-2.V-S2NNrY.png)如果想要关闭文件的查看权限，可以点击**取消分享**，已分享的链接将立即失效：

![](/docs/static/settings-10.DY3GfpkW.png)在**我的分享**中，点击**编辑**快捷管理分享链接：

![](/docs/static/settings-11.DBG87oUR.png)## 其他设置

| 其他选项 | 说明 |
| --- | --- |
| **关注服务号** | 查看官方公众号二维码，关注公众号及时了解产品最新动态 |
| **使用帮助** | 查看帮助文档和常见问题 |
| **用户协议** | 查看服务条款 |
| **隐私政策** | 查看隐私保护说明 |
| **退出登录** | 退出当前微信账号（红色文字） |
